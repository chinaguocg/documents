# Ansible 初识三：Ansible 定义变量

定义变量时，名称不可以是数字或带有空格，但可以使用下划线连接。

使用 vars 关键字来定义变量。 在引用变量时，需要使用“”的格式

使用 vars_files 变量来调用变量文件。

关于主机和主机组变量

[jess@bogon project]$ cat inventory
[qf2]
node2 username=xiaoqianfeng

[nodegroup:children]
qf2

[nodegroup:vars]
group_username=daqianfeng

[qf2:vars]
group_username2=daqianfeng2

关于事实变量

关于事实变量（Fact）
定义： 在目标设备中检索到的各种软/硬件信息。
当每次运行playbook时，ansible会自动去收集目标设备中的事实变量。 或者使用setup模块手动收集。


old:    ansible_distribution_version == 7.8
new:   ansible_facts[‘distribution_version’]  == 7.8   等价于  ansible_facts.distribution_version  == 7.8

ansible_facts[‘dns’][‘namerservers’] 
ansible_facts.dns.nameservers

{{  ansible_facts[‘ens33’][ ‘ipv4’][‘address’]  }}

关于捕获命令的输出
使用register捕获命令的输出到某个变量中。
使用debug模块调试变量内容
