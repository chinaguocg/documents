# 防火墙

iptables和firewalld都属于用户层面的工具，调用的其实是底层。

底层是Netfilter网络过滤模块。

防火墙实现的是包过滤。控制转发。

## firewalld

命令行管理防火墙

```shell
firewall-cmd  

firewall-cmd  --permanent  --add-service=ftp 

firewall-cmd  --reload  
```
