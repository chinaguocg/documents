# 文档项目进阶

## 提交项目代码

在工程目录`documents`下，[管理远程仓库-GitHub官方文档](https://docs.github.com/cn/enterprise-server@2.22/get-started/getting-started-with-git/managing-remote-repositories#changing-a-remote-repositorys-url) 或者参考 [怎么把别人的git换成自己的](https://worktile.com/kb/ask/242699.html) ，变更为自己的仓库地址后再执行如下命令

- 拉取最新代码

```
git pull
```

- 提交变更代码至缓存

```
git add . 
```

- 提交至本地仓库

```
git commit -m “注释部分” 
```

- 推送至远程仓库`master`分支

```
git push -u origin master
```
